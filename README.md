

# 관광지 추천 앱

## 프로젝트 설명

● 공공 API를 이용하여 대한민국의 관광지를 추천합니다.

● 선택한 지역과 관광지 카테고리로 관광지를 추천합니다.

● 현재 위치를 기반으로 주변에 있는 관광지들을 소개합니다.

● 즐겨찾기를 통해 가고싶은 관광지를 저장할 수 있습니다.
